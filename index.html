Portfolio

distracer.io - Browser distributed ray tracer

JavaScript Node.js Browserify HTML5 Canvas Bower Grunt

Website

CrowdProcess, a browser powered distributed supercomputer, launched a call for developing applications using their technology. I've decided to take the challenge and create a distributed ray tracer for generating photorealistic images using their platform.

The rendering engine is written in javascript and I built the webapp using browserified node.js to call their API and push results into a HTML5 canvas. I also used bower package manager together with grunt to build the app.

The project had more then 5000 unique visitors in the first 48 hours once we reached the 2nd page on YCombinator's hacker news. It was a really exciting project.

Nestible - Financial advisor fact-finder

JavaScript Node.js MongoDB Angular.js Twitter Bootstrap

Screenshots

My client wanted a MVP fact-finder to gather relevant changes in the financial life of their customers. He also wanted a back-office to give to Financial Advisors the ability to see the answers of the fact-finding and understand if their intervention is required or not.

I designed and built the front-end using Angular.js with Twitter Bootstrap with responsive design and implemented the backend using Node.js with MongoDB.

My client used the MVP to present his product to several Financial Advisors in the UK and most of them shown interest in buying and investing in his product.

Guinness - Beer drinking job contest

Groovy Grails PostgreSQL Facebook API

Screenshots

My client wanted a web app to receive and manage job applications for a beer drinking position on Guinness (advertisement campaign). That web app should integrate as a tab on Guinness Portugal facebook page, ask people to like the page and invite friends by email for the job application. Only those who received an email invitation from a friend could apply for the job after giving a like on the page.

I developed the facebook tab and the back-office to manage the job applications using Grails and PostgreSQL.

The campaign was a complete success, in a week they acquired 80 000 likes on the facebook page and had about 10 000 job applications.

Base 2 - Portuguese Public Procurement Portal

Java Spring Hibernate WSDL Shibolleth ZK Tomcat Maven

Screenshots

My client asked my team to develop an updated version of the Portuguese Public Procurement Portal. It's a complex system with many integrations with other systems. That integration is done through the Public Administration Interoperability Platform - iAP, an asynchronous Enterprise Service Bus connecting all of these entities. The information about the contracting procedures is made public through a portal while another portal, only available to contracting authorities, is used to gather complementary information on the contracting procedures.

My responsibilities in this project started with the project skeleton creation (build tool, dependency management, base DAO and service classes) but also database and security configuration using Java and Spring Framework. I designed and developed most of the private portal using ZK Framework and also worked in the integration of the asynchronous Web Services.

The project was a success and was distinguished with a CIO Awards 2013 prize.
